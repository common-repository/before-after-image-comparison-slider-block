A Wordpress plugin that show of two image before after comparison.


== Description ==

It's a simple Gutenberg blog plugin that allows you to easily display a comparison slider of two images. It is very easy to use and does not require any technical knowledge to the end users.


Features include:

* Fully Customizable**: All the options you need to arrange the showcase to your liking are available here.


== How to use ==
- First install the Before After Image Comparison Slider plugin
- Add the Before After Image Comparison Slider from the block category called "Widgets" in Gutenberg editor.
- You can add or change the images from the block itself.
- you can also change the before & after text from the block itself.
- Enjoy!
